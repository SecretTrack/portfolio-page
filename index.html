Projects

Magic The Gathering Colors Project

This project is a dynamic web application that showcases the different colors in Magic: The Gathering (MTG). Users can explore each color's unique traits, abilities, and common creatures through an interactive interface. Built with HTML, CSS, and JavaScript, it features a responsive design and engaging hover effects, allowing for an immersive experience in the world of MTG.

Weather API Project

A responsive weather application that provides real-time weather information for cities worldwide. Users can quickly check temperature, humidity, wind speed, UV index, and weather conditions by entering a city name. Built with HTML, CSS, and JavaScript, the app leverages the WeatherAPI to fetch accurate and up-to-date data. The clean, intuitive interface ensures ease of use on both desktop and mobile devices, making it easy for users to stay informed about current weather conditions.
